# Results Overview

This folder contains the plot outputs generated by the pairs trading backtest for the 2023 to 2026-07-10 period.

## Files

- `Backtest Summary for Candidate Pairs.png`

  - Shows overall performance summary across candidate pairs evaluated for this date range.
  - Typically includes Sharpe distribution, Sharpe vs CAGR, and Sharpe vs drawdown.

![Backtest Summary for Candidate Pairs](<Backtest Summary for Candidate Pairs.png>)


- `Top 10 Pairs by Sharpe.png`

  - Displays the top 10 pairs ranked by Sharpe ratio in this backtest run.
  - This is the main result used to compare pair quality.

![Top 10 Pairs by Sharpe](<Top 10 Pairs by Sharpe.png>)


- `Pair1 Plots.png`, `Pair2 Plots.png`, `Pair3 Plots.png`
  - Each file is a dedicated chart for one of the top three pairs.
  - Plots include:
    - cumulative strategy return,
    - drawdown curve,
    - residual spread with entry/exit markers for long and short trades.

![Pair1 Plots](<Pair1 Plots.png>)

## Notes

- Entry and exit markers show when the strategy opened or closed positions based on z-score thresholds.
- Trade performance is evaluated using the fixed training-set hedge ratio applied to the test cohort.
- Use these files to inspect whether the top pairs behave stably and whether the strategy timing looks sensible.
