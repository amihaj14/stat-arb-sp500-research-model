# Results Overview

This folder contains the plot outputs generated by the pairs trading backtest for the 2019 to 2026-07-10 period.

## Files


`Backtest Summary for Candidate Pairs.png`

- Shows the overall performance summary for the candidate pairs evaluated in this run.
- Includes distributions of Sharpe ratios, Sharpe vs CAGR, and Sharpe vs drawdown.

![Backtest Summary for Candidate Pairs](<Backtest Summary for Candidate Pairs.png>)


- `Top 10 Pairs by Sharpe.png`

  - Displays the top 10 candidate pairs ranked by Sharpe ratio.
  - Useful for quickly identifying the highest-performing pairs.

  ![Top 10 Pairs by Sharpe](<Top 10 Pairs by Sharpe.png>)



- `Pair1 Plots.png`, `Pair2 Plots.png`, `Pair3 Plots.png`
  - Each file corresponds to one of the top three pairs.
  - The plots include:
    - cumulative return for the pair strategy,
    - drawdown over time,
    - residual spread with long/short entry and exit markers.

![Pair1 Plots](<Pair1 Plots.png>)

## Notes

- A higher Sharpe ratio indicates a more attractive risk-adjusted return.
- Total return values are decimal returns (e.g. `1.18` means `118%`).
- Entry/exit markers are placed where the strategy entered or exited long/short positions.
